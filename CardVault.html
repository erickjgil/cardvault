Cloud Sync
Syncs your entire catalog across every device. Free for up to ~5,000 cards.

Setup (2 min):
1. Go to supabase.com → New project
2. Settings → API → copy Project URL and anon key
3. Run the SQL below, then paste keys here
SQL to run in Supabase:
create table cards ( id text primary key, data jsonb not null, img_url text, updated_at timestamptz default now() ); alter table cards enable row level security; create policy "public access" on cards for all using (true) with check (true);
